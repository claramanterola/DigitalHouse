EduAdvisor

PROYECTO

Eduadvisor es un proyecto para acompañar en el proceso de orientación a las personas que están buscando dónde y qué estudiar.
Nuclea a universidades y entidades educativas y las contacta directamente con estudiantes interesados.

SOLUCIÓN

Crear una PWA con un perfil que permita generar búsquedas en base a preferencias especificadas por el usuario.
Test vocacional gratuito, preferencias y resultados con los datos de las instituciones.
Contacto directo con las mismas desde la aplicación y seguimiento post-contacto.
