Samsung Catalog

PROYECTO

Desde Samsung Argentina, tenían la necesidad de crear una plataforma para consumidores finales y vendedores de sus productos. Por un lado, la necesidad del cliente de tener que conservar la factura y la garantía, luego de realizada la compra. Por el otro, para los vendedores tener una herramienta de venta y consulta de la cartera de productos. Muchos de los productos de Samsung poseen garantias por años y para realizar cualquier reclamo es obligatorio presentar el ticket de compra, así como la garantia impresa para poder asegurarse la legitima originalidad. De esta forma, es posible rastrear la fecha y lugar de compra y obtener fácilmente los datos del comprador, del vendedor y los detalles técnicos del producto.

SOLUCIÓN

El desarrollo de una aplicación que funcione como catálogo promocional de productos y la función de poder guardar garantias o facturas en forma digital para su uso futuro. Se integran iistintas funcionalidades en base al tipo de usuario: cliente o vendedor.
Para el consumidor final:
- Catálogo de productos sin venta online, solo consulta (la compañía ya posee su plataforma dedicada a e-commerce).
- Sección para guardar documentos de productos (garantias, tickets, etc).
- Sección para descargar información de productos: manuales de uso, fichas técnicas, especificaciones, imágenes en alta definición, videos, etc.
- Vía de comunicación para promociones y descuentos exclusivos.
Para el vendedor oficial:
- Material de consulta de las características y funciones de los productos.
- Plataforma de Capacitación y Encuestas.
- Vía de comunicación y Sistema de recompensas y sorteos.
De forma paralela, surje la necesidad de crear un sistema de iconos para cada categoría, con el fin de describir y comparar las funciones de cada producto (features).
